🧩 JAES

Java AES Encrypt System

AES + RSA によるハイブリッド暗号化ツール

概要

JAES は、AESおよびRSAによるハイブリッド暗号化を実装した ファイル暗号化・復号化ツールです。

ブロックチェーン技術を応用したファイル履歴管理機能を備え、 改ざん耐性と完全性検証を実現します。

特徴

🔒 AES + RSA

データはAESで暗号化し、AES鍵はRSAで保護します。

🧱 ブロックチェーン履歴

暗号化・復号処理を履歴として記録し、整合性を検証できます。

🖼️ PNG出力

暗号化ファイルをPNG形式として保存できます。

☕ Javaバージョン記録

暗号化時のJavaバージョンをファイル内に保存します。

システム要件

  • Java Runtime Environment: Java 25 推奨
  • 対応OS: Windows / macOS / Linux
  • メモリ: 512MB以上推奨
  • 文字コード: UTF-8 推奨

インストール

  1. GitHubリリースページ から最新版の JAES.jar をダウンロードします。
  2. 任意のフォルダに配置します。
  3. Javaがインストールされているか確認します。
java -version

実行方法

通常起動

java -jar JAES.jar

通常起動(64Bitパソコンの場合)

64bitパソコンの場合はJAESlauncher.exeを実行することでも実行可能です。

公開鍵を指定して起動

java -jar JAES.jar "C:\path\to\public.pem"

ポータブルモード

java -jar JAES.jar --portable "D:\JAES\key"

画面クリア無効

java -jar JAES.jar --nocls

出力仕様

  • 暗号化ファイル: .jdec
  • PNG出力: .jpng
  • 復号化時: 元のファイル名へ自動復元

注意事項

  • 秘密鍵を紛失すると復号できません。
  • AppData/JAES/key フォルダは必ずバックアップしてください。
  • Javaバージョン差異がある場合、復号できない可能性があります。
