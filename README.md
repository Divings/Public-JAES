# 🧩 JAES: Java AES Encrypt System

## 概要

**JAES** は、AESおよびRSAによるハイブリッド暗号化を実装したファイル暗号化・復号化ツールです。  
ブロックチェーン技術を応用したファイル履歴管理機能を備え、  
単一ユーザー環境でも高い改ざん耐性と完全性検証を実現します。  

また、暗号化ファイルを **PNG形式** に変換して安全に保存できる機能を標準搭載しています。  
（EncryptSecureDECではWAV出力のみ対応していましたが、JAESではPngEncoderをデフォルト実装しています）

---

## 特徴

- 🔒 **AES + RSA ハイブリッド暗号**  
  データはAESで暗号化され、AES鍵はRSAで保護されます。  
  非対称鍵を用いてもブロックチェーン履歴と連携可能です。  

- 🧱 **ファイル末尾にブロックチェーン履歴を追記**  
  各暗号化／復号処理がブロック（JSON）として記録され、  
  改ざん検出・履歴検証が可能です。  

- 🖼️ **PNGエクスポート対応（ステガノグラフィー）**  
  暗号化ファイルをPNG画像として保存でき、  
  メタ情報に最終更新時刻とチェーンハッシュを埋め込みます。  

- 🔐 **ローカル鍵自動管理（AppData内 key/ フォルダ）**  
  公開鍵・秘密鍵はユーザー環境ごとに自動生成され、  
  外部共有を必要としない完全ローカル運用を実現。  

- 🧮 **圧縮なし・可逆構造**  
  圧縮を行わずバイナリ構造を保持するため、  
  暗号前後の整合性検証が容易です。  

---

## システム要件

- **Java Runtime Environment (JRE)**:  25推奨 
- **対応OS**: Windows / macOS / Linux  
- **メモリ要件**: 512MB以上推奨  
- **推奨文字コード**: UTF-8  
(注意:ファイル名に日本語を含むファイル、もしくはディレクトリに日本語が含まれる場合ファイルを認識できません)
---

## インストール

1. [GitHubリリースページ](https://github.com/Divings/Public-JAES/releases/latest) から最新の `JAES.jar` をダウンロードします。  
2. 任意のフォルダに配置してください。  
3. Java 8 以上がインストールされていることを確認します。  
   ```bash
   java -version
   ```

---

## 実行方法

### 方法1: ダブルクリックで起動
- `JAES.jar` を直接ダブルクリックします。

### 方法2: コマンドラインで起動
```bash
java -jar JAES.jar
```

---

## 使用手順

1. プログラムを起動します。  
2. メニューが表示されるので、以下の操作を選択します。  
   - 暗号化（Encrypt）  
   - 復号化（Decrypt）  
   - PNG変換  
   - ブロックチェーン検証  
3. 暗号化・復号化したいファイルの**フルパス**を入力します。  
   入力を空欄にすると処理を中断してメニューに戻ります。  
4. 操作完了後、出力ファイルのパスが表示されます。  

### 出力仕様

- 暗号化ファイル：  
  元ファイル名に拡張子 `.jdec` を付与。  
- PNG出力：  
  同名ファイルに `.jpng` を付与して保存。  
- 復号化時：  
  元のファイル名・拡張子に自動復元。  

---

## 注意事項

- 暗号化鍵を紛失すると復号はできません。  
  生成された `AppData/key` フォルダの内容は必ずバックアップしてください。  
- ブロックチェーン履歴はファイル末尾にJSON形式で追記されます。  
- ファイル名に日本語を使用する場合はUTF-8対応環境で実行してください。

## トラブルシューティング

### Q. 「ファイルが存在しません」と表示される  
- フルパスに日本語やスペースが含まれる場合は `"..."` で囲って指定してください。  
- USBやネットワークドライブの場合、Javaのアクセス権に注意してください。

### Q. 起動しない  
- Javaがインストールされていないか、PATH設定が不十分な可能性があります。  
- 以下で確認してください：  
  ```bash
  java -version
  ```

### Q. 復号結果が壊れている  
- 入力ファイルが途中で改ざん・破損していないか確認してください。  
- ブロックチェーン検証機能で履歴整合性を確認できます。
